# Getting Started

The first time you open Sculptor, a short setup wizard walks you through three
steps:

1. **Your details** — enter your name and email.
2. **Dependencies** — Sculptor checks that the Claude CLI and git are installed and ready. We also provide some optional
   dependencies. If you want to use Pi as your harness, you can install it here.
3. **Add a repository** — point Sculptor at your first repo.

From there, [create a workspace](workspaces.md), prompt an agent, review its
[changes](changes.md), and open a [pull request](pull_requests.md) when you're
happy.

Your account email is shown later under **Settings > Privacy**.
