# Command Palette

Press `Cmd+K` anywhere in Sculptor to open the command palette — a searchable
list for getting around without the mouse.

Start typing to filter, use the arrow keys to move, `Enter` to run, and `Esc` to
close.

From it you can:

- Switch between workspaces and agents, or create new ones
- Show or hide panels
- Jump to a file
- Open settings or help, or toggle the theme

It's the fastest way to navigate when you have several workspaces and agents
open.
