• Name title first ("Untitled workspace" → fills in), then the three options as stacked pills: Repo · Branch · Source.

• Branch = the generated/target branch with a ⇄ regenerate action; Source is a selectable pill defaulting to origin/main.

• Title + pills are one vertically-centered group; the accent/blue Create button is pinned to the bottom.

• No "Name" heading, no first-prompt field, no "Keep open" toggle (mobile).

• Page base --gray-2 (matches the workspace page); pills on --gray-3.
