Sculptor Mobile — Mocks

The agreed mobile direction. Real Lucide icons, warm "sand" light theme applied on mobile only. Two component pages (Foundations, Chat) and a Flows page driven from an already-in-progress chat: open the workspaces drawer, view changes → review-all, open the terminal, switch agents via the pager dots under the chat input. Mocks only — see spec.md for the implementation approach.

bg #FAE8CA accent #F50D00 icons Lucide vibe Claude mobile

Flows

Assume an agent is already working in a workspace. Everything starts from the chat. The home screen below is interactive — tap the ☰ (drawer), the changes chip, the ⋮ (jump menu), or the agent dots. Each flow is then broken into steps.

Default Landing · new workspace (the screen you open to)

The default screen. One real input — the workspace name. Branch is always origin/main and repo are kept subtle; no recents or shortcut chrome.
